# 期刊数据库 2025

EI核心期刊 + 2025 JCR分区（2024 JIF）+ 2025中科院分区 合并查询数据库

**🔗 在线查询：https://wenquan0816.github.io/journal-database/**

## 数据来源

| 数据库 | 版本 | 期刊数 |
|--------|------|--------|
| 科睿唯安 JCR | 2025年（2024 JIF） | 22,249 |
| 中科院分区 | 2025年 | 21,772 |
| EI核心期刊 | 最新版 | 4,208 |
| **合并后总计** | | **23,846** |

## 功能

- 🔍 实时搜索期刊名称、ISSN、学科
- 筛选 JCR 分区（Q1–Q4）
- 筛选中科院分区（1–4区）
- 筛选数据来源（三库全收录 / 含EI / 含JCR / 含中科院）
- 筛选 Top 期刊
- 点击列标题排序（JIF、分区等）
- 分页浏览，每页50条

## 字段说明

| 字段 | 说明 |
|------|------|
| 期刊名称 | 期刊英文名（大写） |
| ISSN / EISSN | 印刷版 / 电子版 ISSN |
| 2024 JIF | 2024年影响因子 |
| JCR分区 | Q1–Q4 |
| 中科院分区 | 1–4区 |
| EI收录 | 是否被EI核心收录 |
| Top | 是否为中科院Top期刊 |
| OA | 是否为Open Access |
| JCR学科类别 | WOS学科分类 |
| 数据来源 | JCR / CAS / EI 组合 |
